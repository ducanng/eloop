- ## ⌨️ Technology used in the project
  - ### **Back-end, Database**
     - NodeJS (ExpressJS).
     - Hosting: Heroku server
     - Cloud Server: ClearDB Mysql.
   - ### **Front-end**
     - Boostrap
     - Xây dựng UI với Css, Scss.
   - ### **Tool Team Work**
     - Git, Github.

- ## 📦 Installation
  ```
  > git clone https://github.com/ducanng/eloop.git
  > cd eloop
  > npm i
  ```

- ## 🔨 Run project
  - Convert scss to css
  ```
  > npm run watch
  ```
  - Run web application
  ```
  > npm start
  ```
